# User Manual Screenshots

This directory contains screenshots for the User Manual feature.

## Required Screenshots

Place the following screenshot files in this directory:

1. **asset-list.png** - Screenshot of the Asset List module showing the asset table, add/import buttons
2. **repair-request.png** - Screenshot of the Repair Request form or status page
3. **maintenance-scheduling.png** - Screenshot of the Maintenance Scheduling interface
4. **asset-scanner.png** - Screenshot of the Asset Scanner with QR code scanning interface
5. **laboratory-login.png** - Screenshot of the Laboratory Login/Logging interface

## Screenshot Guidelines

- **Format**: PNG or JPG (PNG recommended for better quality)
- **Resolution**: 1920x1080 or similar high resolution for clarity
- **Content**: Capture the main interface of each module showing key features
- **File Size**: Try to keep under 500KB per image (compress if needed)

## Fallback Behavior

If a screenshot is not found, the user manual will display a placeholder with instructions to add the image.

## Taking Screenshots

1. Navigate to each module in your system
2. Use Windows Snipping Tool (Win + Shift + S) or similar screenshot tool
3. Capture the main interface showing key features
4. Save with the exact filename listed above
5. Place in this directory

## Testing

After adding screenshots:
1. Clear browser cache
2. Open the User Manual (book icon in header)
3. Navigate through each slide to verify images display correctly
